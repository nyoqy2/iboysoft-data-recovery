## 几个常见问题，提前给你答案

**Q：买了一年版，到期后还能用吗？**
能继续用，但不能再激活新设备。一年版本质是注册码有效期一年，到期后已激活的电脑可以继续使用软件本身，但更新和换机服务会受影响。所以长期使用者我更推终身版。

**Q：BitLocker 加密盘没密码能恢复吗？**
不能。这是安全机制决定的，任何宣称"无密码恢复 BitLocker"的工具都不可信。赤友需要在你能提供密码或 48 位恢复密钥的前提下才会对加密分区做深度检索。

**Q：Mac 上 T2 芯片和 M 系列机型的内置盘能恢复吗？**
T2 及 Apple Silicon Mac 的内置 SSD 启用了硬件级加密，从 macOS Recovery 模式启动后可以走恢复流程；但如果是已抹掉且未备份的内置盘，硬件加密机制下基本无解——这是所有数据恢复软件共同限制，不是赤友的问题。外接盘、移动硬盘、SD 卡不受此影响。

**Q：激活码能换电脑吗？**
支持重装系统，每年有两次新旧电脑换机服务，联系开发商重置即可。

**Q：试用版扫到了文件，买正式版还要重新扫吗？**
不用，把试用版用激活码激活成正式版，已扫描的结果可以直接恢复，不用重跑一遍——这点设计挺贴心，毕竟深度扫描动辄几十分钟。
